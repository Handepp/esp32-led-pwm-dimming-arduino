# ESP32 LED PWM Dimming

Control LED brightness using ESP32 hardware PWM (LEDC) with smooth fading effect.

---

## 🚀 Features

- Hardware PWM using ESP32 LEDC
- Smooth LED fading effect
- Non-blocking timing using `millis()`
- Serial brightness monitoring

---

## 🔌 Wiring

GPIO 2 ---- Resistor ---- LED ---- GND


---

## ▶️ How to Use

1. Open `main.ino` in Arduino IDE  
2. Select ESP32 board  
3. Upload the code  
4. Observe LED fading smoothly
